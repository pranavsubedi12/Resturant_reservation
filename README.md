# 🍽️ Namaste Nepali Kitchen

A modern full-stack restaurant reservation and management system built with the MERN Stack.

## 🚀 Features

### Customer Features
- Responsive restaurant landing page
- Online table reservation form
- Menu showcase section
- Mobile-friendly UI

### Admin Features
- View all reservations
- Create reservations
- Update reservation status
- Delete reservations

### Backend Features
- RESTful API
- CRUD operations
- MongoDB integration
- Backend validation

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

# 📦 Installation

## Clone Repository

```bash
git clone <your-repository-url>
cd Namaste-Nepali-Kitchen
```

## Install Backend

```bash
cd Backend
npm install
npm run dev
```

## Install Frontend

```bash
cd Frontend/restro
npm install
npm start
```
